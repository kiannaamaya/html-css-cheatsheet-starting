Semantic Element Tags Reference

Semantic Tags

Tag Name Description
<header> Header Specifies introductory content
<nav> Navigation Bar Specifies a set of navigation links
<main> Main Specifies main content of page
<footer> Footer Specifies footer of page
<section> Section Specifies generic standalone section
<article> Article Specifies independent, self-contained content
<aside> Aside Specifies content indirectly related to surrounding content
